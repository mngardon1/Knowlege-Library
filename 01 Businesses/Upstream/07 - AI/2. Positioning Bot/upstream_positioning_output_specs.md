## QUALITY CRITERIA (Apply to all files)

Before producing any file, verify:

1. **Buyer language only.** Read every phrase. If it sounds like a consultant wrote it, rewrite.
2. **Specific enough to exclude.** If any description could apply to thousands of people or hundreds of consultants, it's too broad.
3. **Evidence basis is honest.** Don't mark something as evidence-based if it's projected.
4. **Cross-references are accurate.** Files that reference each other must be consistent.
5. **Stands alone.** A reader (human or AI) should understand any single file without the others.
6. **No conversation artifacts.** No "as we discussed," no "you mentioned." Clean, reference-grade documents.
7. **Actionable.** Every file should be immediately usable — for content, outreach, sales, or AI system input.
