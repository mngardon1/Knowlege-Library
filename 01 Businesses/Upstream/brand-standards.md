## 2. COLOR PALETTE

### Primary
| Name | Hex | Use |
|------|-----|-----|
| Brand Green | `#0A9B6E` | CTAs, highlights, accent text, links |
| Near Black | `#1A1A1A` | Headlines, logo wordmark, body text |
| Warm Off-White | `#F9F9F7` | Page/doc backgrounds |

### Secondary
| Name | Hex | Use |
|------|-----|-----|
| Deep Green | `#087A57` | Hover states, deep accents |
| Mid Gray | `#6B7280` | Nav, secondary text, labels, captions |
| Light Gray | `#E5E7EB` | Dividers, borders, table lines |
| White | `#FFFFFF` | Button text, card backgrounds |

### Rules
- `#0A9B6E` is the ONLY accent color — never substitute blue, teal, or other greens
- Backgrounds are always Warm Off-White or White — never dark mode
- Never use red, orange, or yellow as accents
- Text on green buttons is always White — never black

## 3. TYPOGRAPHY

### Fonts
- **Web:** Inter (Google Fonts)
- **Documents/fallback:** Arial
- **Monospace (code/hex values):** Courier New

### Type Scale (web)
| Level | Size | Weight |
|-------|------|--------|
| Hero | 56–72px | Bold |
| H1 | 40–48px | Bold |
| H2 | 28–32px | Bold |
| H3 | 22–24px | Bold |
| Body | 16–18px | Regular |
| Label/Caption | 12–14px | Bold, ALL CAPS |
| Button | 16px | Bold |

### Rules
- Headlines: Sentence case only — never ALL CAPS for headlines
- Labels and section markers: ALL CAPS with letter spacing
- Never bold body text for emphasis — rewrite the sentence instead
- Never more than 2 font weights in a single layout

## 4. VOICE & TONE

### Four words
**Grounded. Precise. Honest. Direct.**

### On-brand ✓
- "You're good at your work. That's not the problem."
- "Positioning is the upstream problem. Solve it first."
- "You're not lazy. You're misaligned."
- "Diagnosis is the sale."
- "From 1 client to a pipeline you control."

### Off-brand ✗
- "Unlock your full potential and 10x your results!"
- "This game-changing program will transform your life!"
- "Start your hustle journey today!"
- "Limited time offer — act now!"
- Anything with: hustle, grind, crush it, kill it, 10x, game-changer, guru, hack

### Tone rules
- Calm, not urgent — never manufacture scarcity or pressure
- Diagnostic before prescriptive — show what you see before offering a solution
- Precise over clever — if a sentence is clever but vague, cut it
- Confident, not arrogant — state things directly, no hedging with "perhaps" or "might"
- Human, not guru — write to one person, not a stadium
- No passive income language — explicitly rejected in all program messaging
- No "transform your life" — use specific outcomes instead

## 5. CONTENT SYSTEM

### Mandatory sequence — no exceptions
```
Pain → Desire → Insight → Framework → CTA
```
Never skip, reorder, or merge steps.

### Content stages
| Stage | Audience State | CTA Type |
|-------|---------------|----------|
| **Break** | Awareness, permission, small experiments | Low friction, no commitment |
| **Become** | Identity, standards, future vision | Reflection, vision exercises |
| **Build** | Validation, offers, resonance | Programs, calls, offers |

**Critical rule:** Never push Build CTAs from Break content.

### Framework rules
- Use only canonical frameworks from the Knowledge Library
- One framework per piece of content maximum
- Introduce every framework by naming the problem it solves first
- Never teach a framework fully inside content — frameworks stabilize insight, they are not the insight

## 8. DOCUMENT & ARTIFACT STANDARDS

### When building docs
- Page size: US Letter (8.5" × 11")
- Margins: 1 inch all sides
- Primary font: Arial (Inter if web-rendered)
- Accent color: `#0A9B6E` for headers, highlights, key lines
- Background: White or `#F9F9F7`
- Tables: use `#E5E7EB` borders, `#EFF6FF` or `#F9FAFB` for shaded rows
- Never dark backgrounds in documents

### When building HTML artifacts
- Font: Inter via Google Fonts
- Accent: `#0A9B6E`
- Background: `#F9F9F7`
- Text: `#1A1A1A`
- Secondary text: `#6B7280`
- Buttons: `#0A9B6E` background, `#FFFFFF` text, rounded corners

### Section hierarchy in artifacts
1. Label (ALL CAPS, gray, small)
2. Heading (bold, dark, large)
3. Body (regular, dark gray)
4. Supporting detail (smaller, mid gray)
